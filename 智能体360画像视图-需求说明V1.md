# 智能体360画像视图-需求说明

## **1.页面定位**

需要在原「智能体信息详情页」（[统一台账中心-需求说明文档](https://app.notion.com/p/d7c083fb617083c0812a017dc38f93bc?pvs=21) 2.2，本次不改动）新增「360 画像视图」，作为进入详情页后的默认视图。视图以单个智能体为中心，聚合呈现基本 / 技术信息、关联资源拓扑、准入评测、运行监测四大板块，异常连接醒目提示、各板块可下钻查看明细；用户可随时切回原详情页查看完整结构化档案。

## 2.大屏布局与风格说明

「360 画像视图」以**数据大屏**形式呈现，采用深色科技风（深蓝底色、蓝青主色调、发光描边、卡片分区），风格参考下图。顶部为标题栏与一行核心状态卡片（智能体编号 / 名称 / 版本 / 风险分级 / 运行状态），便于快速定位；主体采用「四角环绕 + 中心拓扑」的五模块布局，异常连接与故障醒目高亮，各模块支持点击下钻查看明细。

## 3.模块说明

#### 3.1 基本信息模块（大屏左上角）

聚合呈现该智能体的身份、归属与备案档案信息。

| **字段** | **字段说明** | **字段来源** | **呈现形式** | **交互说明** |
| --- | --- | --- | --- | --- |
| 智能体编号 | 唯一编号，格式「科室编号-准入顺序号」（如 0001），系统自动生成 | 接入中心 | 纯文本 | 点击文本复制编号，提示「复制成功」 |
| 智能体名称 | 智能体名称，限 2–20 字符 | 接入中心 | 纯文本 | — |
| 智能体版本 | 版本号（如 1.0 / 1.1 / 2.0 / 2.1） | 接入中心 | 标签 | — |
| 所属科室 | 科室代码 + 科室名称 | 接入中心 | 纯文本 | — |
| 诊疗环节 | 导诊分诊 / 预问诊 / 预约挂号 / 辅助检查 / 辅助诊断 / 辅助治疗 / 住院 / 手术 / 其他 | 接入中心 | 标签 | — |
| 功能描述 | 智能体工作内容、服务对象、输入信息与输出结果，限 500 字 | 接入中心 | 多行文本，超长省略 | 悬停 / 点击查看完整内容 |
| 智能体来源 | 自研 / 第三方 / 合作研发 | 接入中心 | 纯文本 | — |
| 供应商名称 | 供应商全称，限 2–30 字符 | 接入中心 | 纯文本，超 20 字省略 | 悬停展示全称 |
| 技术联系人 / 联系方式 | 技术联系人姓名（限 2–10 字）与 11 位手机号 | 接入中心 | 纯文本 | 点击复制联系方式 |
| 创建时间 / 更新时间 | 智能体创建与最近更新时间（格式 YYYY-MM-DD HH:mm:ss） | 接入中心（系统） | 纯文本 | — |
| 风险分级 | 高度关注 / 中度关注 / 一般关注 | 台账中心（风险分级评定） | 彩色标签（红 / 橙 / 蓝） | 点击下钻至「风险分级详情页」 |
| 运行状态 | 在线 / 离线 / 更新 / 禁用 / 异常，实时同步 | 运行监控中心 | 状态灯 + 文本 | 异常时悬停显示异常原因 |
| 备案材料（产品说明书等） | 备案 PDF 文件，单文件 ≤30MB | 接入中心 | 「查看 / 下载」图标 | 点击在线预览 / 下载 |

#### 3.2 技术信息模块（右上角）

技术信息按注册时选择的接入方式动态展示对应字段：API 接入为黑盒端到端（填写接口地址 + API key）；SDK / OTel 接入为白盒埋点（由平台自动签发平台 URL + 平台密钥 key 并生成埋点代码）。切换接入方式后下方字段联动变化；密钥类字段默认掊码 / 密文展示，点击 icon 可查看明文。

| **字段** | **字段说明** | **字段来源** | **呈现形式** | **交互说明** |
| --- | --- | --- | --- | --- |
| 接入方式（公共） | API 接入 / SDK 接入 / OTel 接入，选择后动态展示对应子字段 | 接入中心 | 标签 | 切换接入方式后下方字段联动变化 |
| 接口地址（API 接入） | API 接入方式下填写的接口 URL 地址（http / https 开头） | 接入中心 | 纯文本 | 点击复制；支持「测试连接」验证连通性 |
| API key（API 接入） | API 接入方式下的鉴权密钥，限 8–64 字符 | 接入中心 | 默认密文（********） | 点击 icon1 切换显隐、icon2 复制 |
| 平台 URL 地址（SDK / OTel 接入） | SDK / OTel 接入方式下平台自动签发的平台 URL 地址 | 接入中心 | 纯文本 | 点击复制 |
| 平台密钥 key（SDK / OTel 接入） | SDK / OTel 接入方式下的鉴权密钥，点击【获取 SDK / 获取 OTel】由平台自动签发 | 接入中心 | 掊码密文（如 sk-****） | 点击 icon 查看明文 / 复制 |
| 埋点代码（SDK / OTel 接入） | 根据平台 URL 地址与密钥 key 自动生成的埋点代码 | 接入中心 | 代码块 | 点击复制 |
| 接入时间（公共） | 智能体接入时间（格式 YYYY-MM-DD HH:mm:ss） | 接入中心 | 纯文本 | — |
| 备案材料（技术规格书 / 其他材料） | 技术材料 PDF，单文件 ≤30MB；技术规格书含接口地址、认证方式、请求 / 返回参数、数据格式、错误码说明等 | 接入中心 | 「查看 / 下载」图标 | 点击在线预览 / 下载 |

#### 3.3 关联资源拓扑地图模块（大屏中间）

以智能体为中心，环形拓扑呈现其已对接的院内资源及连接健康度。

| **字段** | **字段说明** | **字段来源** | **呈现形式** | **交互说明** |
| --- | --- | --- | --- | --- |
| 智能体虚拟形象 | 当前智能体的中心锚点 | 台账中心 | 居中虚拟形象 / 图标 | 拓扑中心节点 |
| 资源名称（资源节点） | 已对接院内资源（如 HIS / LIS / PACS / EMR / 药品库 / 知识库 / 医保系统 / 满意平台等） | 医院资源管理中心 | 环形拓扑节点（图标 + 名称） | 点击节点下钻资源明细 |
| 对接状态 | 正常 / 异常 | 运行监控中心 | 连线与节点颜色：正常蓝青「已连接」，异常红色高亮 / 闪烁 | 点击异常连线 / 节点查看异常详情 |
| 资源负责人 / 联系方式 | 对接资源的负责人及联系方式 | 医院资源管理中心 | 下钻弹层文本 | 点击节点下钻展示 |
| 对接方式 | 各资源的对接方式 | 医院资源管理中心 | 下钻弹层文本 | 点击节点下钻展示 |
| 连接统计 | 已连接 / 异常连接数量（单位：个） | 运行监控中心 | 数字角标 | — |

#### 3.4 准入评测模块（左下角）

聚合呈现该智能体准入评测的总分、维度得分与历史趋势。

| **字段** | **字段说明** | **字段来源** | **呈现形式** | **交互说明** |
| --- | --- | --- | --- | --- |
| 评测结果总分 | 最近一次准入评测综合总分（满分 100 分） | 准入评测沙盒（评测结果报告） | 环形进度图 + 数值 + 星级 + 评级（如「优秀」） | 点击下钻至评测结果报告页 |
| 最近评测日期 | 最近一次评测完成日期（格式 YYYY-MM-DD） | 准入评测沙盒（评测结果报告） | 纯文本 | — |
| 安全性评测各维度得分 | 最近一次评测的安全性各维度得分明细（满分 100 分） | 准入评测沙盒（评测结果报告） | 横向条形图 | 悬停显示维度名称与分值 |
| 多次评测得分趋势 | 历次评测总分趋势 | 准入评测沙盒（评测结果报告） | 折线图（X 轴为评测日期） | 悬停显示某次评测日期与分值 |
| 评测结果报告 | 完整评测报告文件 | 准入评测沙盒（评测结果报告） | 「查看 / 下载」图标 | 点击在线预览 / 下载 |

#### 3.5 运行监控模块（右下角）

聚合呈现该智能体的调用、运行健康度与告警 / 故障情况。

| **字段** | **字段说明** | **字段来源** | **呈现形式** | **交互说明** |
| --- | --- | --- | --- | --- |
| 总调用量 | 智能体接入运行后的被调用次数（单位：次） | 运行监控中心 | 数值卡片 | 点击下钻至运行监控业务监控页 |
| 平均在线持续时长 | 各智能体处于"在线"状态的累计时长均值（单位：小时） | 运行监控中心 | 数值卡片，支持日 / 周 / 月切换 | 切换统计周期；点击下钻至运行监控状态监控页，查看各智能体在线时长明细 |
| 平均异常持续时长 | 异常智能体从转入"异常"到恢复（或至当前时刻）的平均时长（单位：分钟） | 运行监控中心 | 数值卡片，支持日 / 周 / 月切换 | 切换统计周期；点击下钻至运行监控状态监控页，查看当前异常智能体清单及各自持续时长 |
| 告警次数 | 智能体接入运行后告警次数（单位：次） | 运行监控中心 | 卡片，橙色数值 | 点击下钻至告警中心页查看明细 |
| 告警次数趋势图 | 智能体接入运行后日/周/月的告警次数趋势（单位：次）；均统计12个周期 | 运行监控中心 | 折线图，支持日/周/月切换，橙色线条与告警次数卡片呼应 | 切换统计周期；悬停数据点显示"日期：X 次"；点击数据点下钻至告警中心页，自动筛选该时段告警明细 |
| 调用量趋势图 | 智能体接入运行后日/周/月的被调用次数趋势（单位：次）；均统计12个周期 | 运行监控中心 | 折线图，支持日/周/月切换 | 切换统计周期；点击下钻至运行监控业务监控页 |

**说明**：以上字段为「360 画像视图」聚合呈现范围，若需查看更完整的结构化档案（基本信息 / 技术信息 / 备案材料 / 已对接资源 / 评测结果等），可切换回原「智能体信息详情页」。
