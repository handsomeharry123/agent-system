# 统一台账中心-需求说明文档

### 模块定位

统一台账中心集中展示全院已审核纳管的智能体核心信息,并提供「台账总览」统计、「台账列表」检索、「智能体信息详情页(画像)」信息维护、「智能体风险分级页」(问卷评定 + 二次确认)等能力,实现智能体全盘掌握与全生命周期台账管理。所有基础信息均根据**接入中心**注册表单导入,台账侧不重复录入。

<aside>
🗺️

**画像与图谱**:智能体信息详情页左侧展示智能体虚拟形象,并在形象周边以关联图谱形式展示其所对接的资源;页面右上角以 Tab 标签页形式集中展示基本信息、技术信息、备案材料、已对接资源信息、评测结果信息五个标签页,各标签页内容互不重复展示。

</aside>

### 系统角色

| 角色 | 数据范围 | 主要职责 |
| --- | --- | --- |
| 管理员(信息科管理员) | 全院所有智能体 | 查看总览统计、检索台账、编辑信息、风险分级评定、禁用、跳转关联模块 |
| 科室用户(科室管理员、科室普通用户) | 本科室智能体 | 查看本科室台账与统计、查看详情 |

### 2.3 智能体风险分级页

风险分级用于对智能体进行医疗风险等级评定。评定结果仅分为 **高度关注 / 中度关注 / 一般关注** 三级,不再区分「初步判定 / 复核判定」。

**操作权限**

- 信息科管理员:可对全院所有智能体进行风险评定操作。
- 科室管理员:仅能看到本科室的智能体,且仅能对自己接入的智能体进行风险评定操作。

**风险分级流程**

1. 用户(科室管理员 / 信息科管理员)在台账列表页或智能体信息详情页点击【风险分级】,进入风险分级问卷页;
2. 填写风险分级问卷(共 7 题,每题单选 A/B/C);
3. 点击【提交】,系统根据判定规则自动评定风险等级,并弹窗展示评定的风险等级(高度关注 / 中度关注 / 一般关注)及判定依据;
4. 用户在弹窗中完成二次确认:点击【确认】,风险等级生效并同步到台账总览页、台账列表页、智能体信息详情页,提示「评定成功」;点击【取消】,返回问卷页可修改后重新提交;
5. 后续如需修改风险等级,可在台账列表页点击【风险分级】入口,重新填写问卷并确认,新结果将覆盖原风险等级。

#### 2.3.1 风险分级问卷

**① 问卷填写**

底部操作【提交】【返回】:点击【提交】进入风险等级评定与二次确认;点击【返回】回到上一页(台账列表页),若有未提交修改则弹出二次确认。

1. 本智能体是否参与临床诊疗决策或治疗过程?
A. 不参与临床诊疗,仅用于健康信息或流程辅助
B. 提供医疗诊疗过程中的重要参考信息或关键技术支持
C. 参与辅助诊断或治疗,直接影响诊疗决策或治疗路径
2. 本智能体的核心功能类型:
A. 健康科普 / 导诊 / 非诊疗支持
B. 医疗信息处理 / 辅助分析工具
C. 临床辅助决策或治疗支持系统
3. 本智能体提供医疗信息支持的深度:
A. 提供通用健康知识或流程信息
B. 提供结构化医疗信息支持(如报告解读、摘要分析)
C. 提供影响临床判断的关键医疗信息或分析结果
4. 当智能体发生错误时的影响:
A. 无临床影响或仅信息误差
B. 影响诊疗效率或可能导致延误
C. 可能导致错误诊疗或严重医疗后果
5. 本智能体应用的医疗场景风险等级:
A. 非医疗或一般健康管理场景
B. 常见病或慢性病管理场景
C. 肿瘤、心脑血管或危重症等高风险医疗场景
6. 本智能体处理的数据类型:
A. 无患者数据,仅公开医学知识
B. 脱敏或匿名医疗数据
C. 真实患者医疗数据(病历/影像/检验等)
7. 本智能体对医疗系统或流程的控制能力:
A. 独立运行,不对接医疗系统
B. 在医疗系统中只读或查询(HIS 等)
C. 可写入系统或触发医疗业务/设备执行

**判定规则**(判断优先级:高度关注＞中度关注＞一般关注)
· 高度关注:Q1–Q7 中任一选 C
· 中度关注:Q1–Q7 中任一选 B
· 一般关注:Q1–Q7 全部选 A

## 三、与其他模块的联动关系

| 数据来源 / 去向 | 联动说明 |
| --- | --- |
| 接入中心 → 台账中心 | 1.智能体编号、名称、版本、所属科室、诊疗环节、来源、供应商、功能描述、技术联系人、联系方式、接入方式(API/SDK/OTel 及其密钥、URL、埋点代码)、备案材料等均取自接入中心注册表单导入,台账侧不重复录入 |

2.总览页的总调用量、异常告警次数、正常运行率等运行指标均取自运行监控中心,卡片可下钻至运行监控业务监控页/告警中心页;
3.台账执行「禁用」后运行监控中心数据相应变化;
列表「更多」可跳转运行监控中心 |
| 医院资源管理中心 ↔ 台账中心 | 1.「已对接资源信息」的资源名称、资源负责人、联系方式、对接方式来源于医院资源管理中心(注册资源页);
2.列表「更多」可跳转医院资源管理中心 |
| 准入评测沙盒 → 台账中心 | 1.智能体经准入评测纳管后进入台账;
2.详情页「评测结果信息」的评测总分、安全性各维度得分、多次评测趋势取自评测结果报告页;
3.列表「更多」可跳转准入评测沙盒 |
